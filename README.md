# Reef environments shape microbial partners in a highly connected coral population

**Kriefall NG, Kanke MR, Aglyamova GV, Davies SW** | *Proceedings of the Royal Society B* (2022)  
DOI: [10.1098/rspb.2021.2459](https://doi.org/10.1098/rspb.2021.2459)

## Overview

This repository contains data and code supporting our study of how reef zone environments structure algal and bacterial communities in *Acropora hyacinthus* corals in French Polynesia. Using 2b-RAD sequencing alongside ITS2 and 16S metabarcoding, we show that reef zones shape symbiont communities at different spatial scales, even within a highly connected coral population where host genetic structure is limited.
